## 🧭 How to Navigate

**☰ Menu button** (top left) — opens and closes the recipe list panel. On mobile it slides over the screen. On desktop it collapses beside the detail view.

**Search bar** (top right) — type any slot name, goal, tag, or author to instantly filter the list.

**Category tabs** — filter by type:

| Tab | Shows |
|-----|-------|
| ⚡ All | Every recipe |
| 🛡️ Armour | Gloves, Boots, Helmet, Body Armour, Belt |
| ⚔️ Weapons | Bow, Wand, Crossbow, Quarterstaff, Spear, Focus |
| 💍 Jewelry | Ring, Amulet, Quiver, Jewel |
| 🥷 Monk | All Martial Artist / Monk specific crafts |
| 💰 Profit | Profit flip recipes |
| 🪙 Budget | Low cost crafts |
| 🏆 BIS | Mirror / endgame tier |
| ❌ Nerfed | Outdated methods — kept for reference |

**Recipe list** — tap any recipe to open the full guide in the detail panel.

**🔍 Base to Search** — expandable section at the top of each recipe showing what mods to look for on the magic base before crafting. Prefix and suffix priorities ranked 1→4.

**Steps** — tap any step to expand the full description. Tap again to collapse.

**Tags** — tap any `#tag` at the bottom of a recipe to filter the list by that tag.
